# 概念細化 3：風的 ramp 衰減參數 + 無的萬金油平衡

---

## 一、風的 ramp：切換衰減 + 時間衰減

### 用戶觀點確認

> 「只有時間衰減等於戰鬥中一直保持滿載，因為怪只在一個階段出現」

正確。TD 的怪物是有限波次，不是持續刷新：
- 有目標時：風一直在打 → ramp 持續累積 → 時間衰減被持續攻擊抵銷 → 等於永遠滿載
- 沒目標時（波次間）：就算 ramp 時間衰減也無所謂，因為下一波還沒開始

**結論：時間衰減在 TD 架構下作用很小。**

主要有效的機制只有一個：**切換目標時的衰減（switch decay）**。

---

### 切換衰減的設計問題：衰減多少？

| 方案 | 切換後保留 | 效果 |
|------|-----------|------|
| A：保留 0%（現況） | 0 | 切換即歸零，普通波次無效 |
| B：保留 50% | 一半 | 清群時維持中等速度，Boss 時能疊滿 |
| C：保留 70% | 大部分 | 幾乎不懲罰，快速清群也能接近滿載 |
| D：保留 100% | 全部 | 等於無懲罰，跟 AOE 一樣強 |

**B（50%）的模擬：**
```
設 ramp cap = 10 層（滿速）
敵人 A 打 6 次 → ramp = 6 層
→ 敵人 A 死，切換到 B → ramp = 3 層（保留 50%）
→ 敵人 B 打 7 次 → ramp = 10 層（上限）
→ 敵人 B 死，切換到 C → ramp = 5 層
```
→ 密集波次中 ramp 從不歸零，普通小怪也能維持中等加速。

**問題：保留比例應該是固定值嗎？**

另一種方案：**每次切換固定扣 N 層（絕對值）**，不是百分比。
```
ramp cap = 10 層，切換扣 3 層
→ 打了 3 層 → 切換後 = 0（打少扣完）
→ 打了 10 層 → 切換後 = 7（保留大部分）
```
→ 「累積越多，保留越多」— 獎勵持續攻擊同一目標的行為。

**推薦：** 切換時扣固定層數（例如 3 層），加上時間衰減（每秒 -1 層）作為次要機制（處理怪群完全消失的空窗）。

---

### 風 ramp 的最終定位

```
核心邏輯：
- 每次攻擊同一目標：+perHit 攻速加成（上限 cap）
- 切換目標：-N 層（固定扣值，不是歸零）
- 每秒不攻擊：-1 層（次要衰減，處理波次間空窗）
```

這樣：
- 普通波次快速清群：每個目標打幾次，切換扣一點 → 維持中等加速 ✅
- 面對 Boss（長時間單體）：ramp 疊滿 → 最高輸出 ✅
- 波次結束後：慢慢衰減歸零，下波重新建立 ✅

## 二、無（⬜）：萬金油的平衡設計

### 問題核心

> 「萬金油太萬用什麼都有 → 獨一檔強」

標準的平衡設計問題：**靈活性本身就是一種強度**，必須有代價。

---

### 無的兩個價值層

**層一：混合塔的「反克制」價值**
無屬性 + 任何元素 = 可建造「沒有元素弱點」的混塔，或者特殊功能組合。
這個價值在**混塔路線**（LV4 雙屬注入）上體現，不需要在純塔上重複體現。

**層二：純塔的定位**
純無（無×無×無）= 不借助任何元素優勢，就是靠基本攻擊力。

---

### 解法：純塔強度 = 「低 DPS + 高經濟回報」

無的純塔**不應該在傷害上追趕其他元素**。其競爭力來自於：

| 資源 | 其他元素純塔 | 純無塔 |
|------|------------|-------|
| 攻擊傷害 | 高（各有特化）| 中等（無特化）|
| 元素優勢 | +30%（克制目標）| 無加成 |
| 元素弱點 | -30%（被克制）| 無弱點（穩定輸出）|
| 金幣收益 | 無 | killGold（擊殺額外金）|
| 被克制風險 | 有（選錯元素）| 無 |

**核心定位：「穩定型 + 經濟回報」**
- 不是最強的傷害，但從不被克制（沒有 -30% 的情境）
- 每次擊殺額外賺金幣 → 讓玩家滾動更多塔/升級

---

### 「萬金油但不破」的核心原則

> 「無的靈活性應該體現在『用什麼元素組合都有無的功能』，而不是『純無本身什麼都做得好』。」

換句話說：
- 純火 × 純火 = 最強火傷害
- 混塔 無×火 = 反克制（無風險）+ 部分火技能 → 傷害比純火低，但更安全
- 純無 = 中等穩定傷害 + 持續金幣收益

**不破的關鍵：純無的 DPS 明顯低於其他元素純塔**，它的「好」體現在金幣複利，不在傷害。

---

### 無的 LV4 → LV5 → LV6 梯度

| 等級 | 技能 | 設計意圖 |
|------|------|---------|
| LV4 | 中等傷害（無特殊）| 穩定輸出，但無法和同級元素塔比傷害 |
| LV5 | 中等傷害 + killGold（小）| 開始有經濟回報 |
| LV6 | 中上傷害 + killGold（大）| 大量金幣滾動，後期複利顯著 |

**LV4 的「無特殊技能」是刻意的**：無系的「特殊」是無弱點 + 反克制混塔可能性，不是技能特效。

---

### 反克制塔（混塔路線）的補充說明

玩家持有無元素 pick 的優勢在混塔路線上：

```
無×火 混塔 = 火的 burn DOT，但傷害元素是「無」（不怕水克）
無×雷 混塔 = 雷的 multishot，但傷害元素是「無」（不怕任何克制）
```

→ 這讓「持有無 pick」在 pick 選擇上有獨特戰略價值：**犧牲高峰傷害，換取無風險出力**。

## 三、六屬定位最終概念版

| 元素 | 定位 | 核心機制 | 代價 |
|------|------|---------|------|
| 🔥 火 | 單體 Boss Killer | DOT + 引爆 無視護甲 | 慢攻速，需要時間堆疊 |
| 💧 水 | 群體控場 | AOE + 緩速（自增效）+ %HP | 傷害分散，單體偏弱 |
| ⛰️ 土 | 穿陣砲擊 | pierce 後排遞增 + 傷害光環 | 攻速慢，稀疏時效益低 |
| 🌪️ 風 | 動量型輸出 | ramp 切換衰減 + 射程光環 | 需要連續作戰才能爆發 |
| ⚡ 雷 | 爆發型清群 | multishot + 射速光環 | 無特殊穿透，依賴數量 |
| ⬜ 無 | 經濟 + 萬用 | killGold + 無弱點 | DPS 低於所有元素塔 |

**代價設計邏輯：**
每個元素都有明確的「什麼情況下是最差選擇」，保證玩家選擇有意義。
