# 概念細化 4：無屬性的第三效果

## 設計限制回顧

純無目前確定的：
- 中等（偏低）傷害
- 無元素弱點（穩定輸出）
- killGold（擊殺賺額外金幣）
- 缺少第三個技能讓玩家感覺「有特性」

第三效果的條件：
1. 不是純 DPS 技能（那些都給其他元素了）
2. 不能讓純無變成「什麼都強」
3. 要跟 killGold 有互動感或互補感
4. 主題符合「無 / 虛空 / 中性」

---

## 候選分析

### 候選 A：unstable（不穩定浮動）

每次傷害 × (1 ± variance)，隨機波動。

**與無的搭配：**
- killGold + unstable = 「高峰時大量擊殺賺大錢，低谷時少賺」，有賭博感
- 主題：「無」= 混沌、不確定性，沒有形態

**問題：**
- 高變異 = 傷害低谷時可能殺不死敵人（違反純塔獨立擊殺原則）
- 感覺像是「隨機性副作用」而非「特性」

**評估：⚠️ 主題合但機制問題大**

---

### 候選 B：lifedrain（生命汲取）

每次對敵人造成傷害，回復基地 HP = damage × pct。

**與無的搭配：**
- killGold = 殺敵賺金幣；lifedrain = 打敵回血
- 一個塔同時提供兩種資源回收（金幣 + 基地 HP）
- 主題：「無」= 虛空汲取，從攻擊中吸收生命力反哺

**特性：**
- 全遊戲唯一有治療效果的純屬塔
- 不增加傷害，但讓玩家更能撐過高壓波次
- 越打越多敵人 = 越多回血，跟密集波次高度協同

**評估：✅ 機制獨特，全屬性唯一，主題合理**

---

### 候選 C：execute（斬殺）

目標 HP < threshold 時，傷害乘以 mult（大幅增傷）。

**與無的搭配：**
- killGold + execute = 「確保擊殺 → 更多 killGold 觸發」
- 補上「中等傷害打不死殘血」的缺口
- 主題：「無」= 冷靜效率，快速終結不浪費一發

**問題：**
- execute 依賴目標已被削血 → 在純塔環境還好（目標進入射程時可能已受傷）
- 但在只有無塔的情境，敵人 HP 不會低 → execute 長時間沒觸發

**評估：⚠️ 跟 killGold 協同但需要前提條件**

---

### 候選 D：permaBuff（永久強化）

每次擊殺此塔永久 +X 攻擊力。

**與無的搭配：**
- killGold + permaBuff = 「殺越多越賺 + 越殺越強」雙複利
- 主題：「無」= 從虛無中成長，沒有先天優勢但能後天積累

**問題：**
- 複利效應在後期可能打破「低 DPS」的設計前提
- 需要嚴格限制 perKill 的 +值，否則 30 波後純無變最強
- 容易因參數難調整而出現「獨一檔」問題

**評估：⚠️ 主題好但平衡風險高**

---

## 推薦：候選 B（lifedrain）

### 理由

| 維度 | 分析 |
|------|------|
| 唯一性 | 全屬性唯一的治療機制，純無有「不可替代感」|
| 不破壞平衡 | 不增加傷害，只回血；不影響殺傷力排名 |
| 與 killGold 互補 | 一個給金幣（進攻資源），一個給 HP（防守資源）|
| 密集波次協同 | 高密度怪群 = 多次 lifedrain = 更多回血，獎勵高強度防守 |
| 主題 | 「無」的虛空汲取感，從攻擊中汲取生命力 |

### 純無塔最終定位

```
定位：「持久型後勤塔」

攻擊：中等穩定傷害，無任何元素弱點
技能：killGold（每殺賺錢）+ lifedrain（每擊回血）
角色：放在高流量通道，一直輸出 → 持續賺錢 + 持續回血

選無的理由：
- 不確定敵人元素 → 無風險穩定輸出
- 想要經濟優勢（金幣複利）
- 基地 HP 快耗盡需要回血
- 混塔路線需要無元素 pick

不選無的理由：
- 想要最高 DPS → 選對應元素
- 敵人元素已知且明確 → 選有克制優勢的元素
```

| 元素 | 核心技能 | LV6 附加 | 「選這個」的理由 |
