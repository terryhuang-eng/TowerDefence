# 概念細化 5：無屬性第三效果重新發想

## lifedrain 排除原因（用戶確認）
召喚怪問題：splitOnDeath / summon 會產生大量子體，即使每次只回 1 HP
→ 一波召喚怪打下來回血量遠超設計值，無法用「小數字」解決。

---

## 重新發想：不限制現有技能

### 方向：「無」的身份核心

「無」= 高效率、無浪費、普世、萬用。
第三技能應該讓玩家感覺「使用無塔有獨特策略感」，而不是「這只是個弱化版的其他元素塔」。

---

## 候選 A：超殺傳遞（Overkill Chain）

**機制：**
當此塔的攻擊傷害超過目標剩餘 HP，多餘的傷害（超殺值）立即傳遞到路徑上下一個敵人。

```
例：目標剩 30 HP，塔打出 80 傷害
→ 目標死亡，多餘的 50 傷害打到下一個敵人
```

**分析：**
| 維度 | 內容 |
|------|------|
| 主題 | 「無」= 不浪費任何傷害，虛空穿透 |
| 與 killGold 協同 | 超殺觸發更多擊殺 → 更多 killGold |
| 策略性 | 高單次傷害的攻擊超殺效果更顯著 |
| 獨特性 | 全屬性唯一的「傷害不浪費」機制 |
| 問題 | 超殺值需要計算（需新增程式邏輯），但不複雜 |

---

## 候選 B：連殺衝刺（Kill Rush）

**機制：**
每次擊殺後 X 秒內攻速加成 +Y，可多層疊加（上限 Z 層）。
X 秒內沒有下一個擊殺，層數清零。

```
例：殺 1 隻 → 攻速 +20%（3 秒）
     3 秒內殺第 2 隻 → 攻速 +40%（重置計時）
     繼續殺 → 攻速最高 +100%
     3 秒無擊殺 → 歸零
```

**分析：**
| 維度 | 內容 |
|------|------|
| 主題 | 「無」= 高效獵手，殺機不斷 |
| 與 killGold 協同 | 殺越快 → 攻速越高 → 殺更快 → 賺更多 |
| 差異於 ramp | ramp = 打同一目標；kill rush = 快速消滅多目標（清群感）|
| 差異於 multishot | multishot = 每 N 攻擊多射幾發；kill rush = 擊殺觸發加速 |
| 問題 | 多敵人波次強，稀疏波次弱（正常，符合設計）|

**Note：** multishot 目前有 killBonus/killDur 機制，但那是「槍管」的額外攻擊；kill rush 是「攻速本身加速」，不同概念。

---

## 候選 C：財富積累（Wealth Scale）

**機制：**
此塔的傷害隨本局累積金幣數成長。每賺 N 金幣，此塔永久 +X 傷害。

```
例：本局已賺 500g → 此塔 +10 傷害
     本局已賺 1200g → 此塔 +24 傷害
```

**分析：**
| 維度 | 內容 |
|------|------|
| 主題 | 「無」= 金錢就是力量，反映玩家的整體經濟策略 |
| 與 killGold 協同 | killGold → 賺更多 → 財富積累加快 → 更強 → 殺更多 → 更多 killGold |
| 獨特性 | 全遊戲唯一與「總金幣數」掛鉤的機制 |
| 策略深度 | LTW 系統的玩家（積極送兵賺 income）會自然讓此塔更強 |
| 問題 | 需要「totalGoldEarned」追蹤（目前已有嗎？需確認）<br>純防守玩家（不送兵）gold 少 → 此塔較弱（可能是懲罰還是獎勵？）|

---

## 候選 D：虛空標記（Void Mark）

**機制：**
攻擊施加「虛空標記」，被標記的敵人死亡時爆炸，對周圍一定範圍敵人造成傷害。

```
標記 → 任意塔擊殺 → 爆炸傷害波及附近敵人
```

**分析：**
| 維度 | 內容 |
|------|------|
| 主題 | 「無」= 虛空留下的印記，死亡也有代價 |
| 與 killGold 協同 | 爆炸造成附加擊殺 → 更多 killGold |
| 差異於 detonate | detonate = burn stacks 觸發；void mark = 死亡觸發 |
| 策略性 | 高 HP 密集怪群效果最好 |
| 問題 | 實作較複雜（死亡 event + 爆炸範圍）<br>與火系 detonate 主題有重疊感 |

---

## 候選 E：中和（Neutralize）

**機制：**
此塔的攻擊「中和」目標的防禦性被動。被中和的目標在一段時間內：
- dodge（閃避）機率降低或消失
- resilient（不屈）累積效果暫停
- fortify（上限）暫時失效

```
概念：「無」不受任何規則約束，攻擊本身讓敵人的特殊能力失效
```

**分析：**
| 維度 | 內容 |
|------|------|
| 主題 | 「無」= 否定存在，讓規則不再有效 |
| 獨特性 | 全遊戲唯一的「反被動」機制 |
| 策略深度 | 特定波次（高閃避/高防禦敵人）中，無塔成為必選 |
| 與 killGold 協同 | 讓其他塔殺不掉的目標變得可殺 → 更多擊殺 → 更多 killGold |
| 問題 | 效果不穩定（無特殊被動的波次中此技能完全無用）<br>設計依賴「對應的特殊敵人」才有意義 |

## 推薦方向

**首選：A（超殺傳遞）**
機制乾淨，主題強，跟 killGold 自然協同，全遊戲唯一。唯一代價是「有時候超殺值很小，幾乎感受不到」。

**次選：C（財富積累）**
與 LTW 系統（送兵賺 income）深度協同，讓「送兵玩家」和「塔防玩家」兩種風格都能受益。需要設定成長上限（例如最多 +50 傷害）。

**B（連殺衝刺）作為附帶考量：**
如果 A 或 C 選定後，B 可以考慮作為 LV6 的額外附加，讓純無 LV6 同時有：killGold + 主技能 + 少量連殺加速。
