# 概念細化 6：無屬性 — 財富積累 + 利息機制

## 確認方向

純無塔的身份定位：**「經濟引擎型」**
- 傷害中等但穩定（無弱點）
- 所有機制都圍繞「金幣」運作
- 選擇純無 = 押注於「越有錢越有力」的複利玩法

---

## 一、財富積累（傷害與當前資金掛鉤）

### 機制設計

**傷害隨「當前持有金幣」動態縮放：**

```
有效傷害 = baseDmg × (1 + currentGold / scalingDivisor)
```

例（scalingDivisor = 200）：
- 持有 0g → 傷害 × 1.0（基礎值）
- 持有 200g → 傷害 × 2.0（雙倍）
- 持有 400g → 傷害 × 3.0（三倍）
- 上限（如 500g 封頂）→ 傷害 × 3.5

### 關鍵設計張力

使用「**當前持有金幣**」而非「總賺取金幣」，產生以下決策：

| 情境 | 效果 |
|------|------|
| 金幣充足、不急著升級 | 傷害高，持續輸出 |
| 花錢升塔後 | 金幣減少，傷害暫時下降 |
| 升完塔後金幣慢慢回收 | 傷害逐步恢復 |

→ **玩家需要選擇「何時升級最划算」**，而不是「越早升越好」。這是其他元素塔沒有的決策維度。

### 需要確認的參數

| 參數 | 考量 |
|------|------|
| scalingDivisor | 決定每塊錢換多少傷害，需比對其他元素塔的 DPS 設計一個「有意義但不過強」的上限 |
| 是否有硬上限 | 建議有（例如最多 × 3.5），防止超後期爆傷 |
| 是否 per-tower 計算 | 每座無塔各自讀取 currentGold（是），多座塔共享同一個 currentGold |

## 二、利息（每波結束依當前資金獲得獎勵）

### 機制設計

```
波次戰鬥結束（進入 pre_wave）時：
bonus_gold = floor(currentGold × interestRate)
```

例（interestRate = 0.05，即 5%）：
- 持有 200g → 獲得 +10g
- 持有 400g → 獲得 +20g
- 持有 800g → 獲得 +40g（若有上限則封頂）

### 與 LTW 系統的協同

```
傳統 income（永久）→ 每波固定收入
利息 income（動態）→ 持有越多、每波額外收入越多
```

兩者疊加：持有大量金幣的玩家每波收入遠高於一般玩家。

### 平衡風險分析

| 風險 | 描述 | 解法 |
|------|------|------|
| 後期爆炸性複利 | W15+ 已有 600g+ 閒置，利息滾動超快 | 設定每波利息上限（例如最多 +50g）|
| 「不花錢」策略 | 純存錢不升塔，靠利息滾 | 財富積累跟 currentGold 掛鉤，不升塔 = 少了新塔的 DPS，存錢策略本身有防守風險 |
| PVP 差距拉大 | 有 利息 的玩家與對手金幣差距越來越大 | 利息來自自身金幣，不影響對手；差距真實存在但同樣受「少升塔」的風險制衡 |

### 觸發條件討論

**方案一：條件是「擁有純無塔在場」即有全局利息**
→ 簡單，任何純無塔觸發一次，不疊加

**方案二：每座純無塔各自計算利息**
→ 多塔複利過強，不推薦

**推薦方案一**：只要場上有純無塔，每波結束觸發一次利息。

## 三、三個機制的完整協同循環

```
killGold（擊殺賺金）
    ↓ 增加 currentGold
財富積累（傷害 ∝ currentGold）
    ↓ 傷害更高 → 殺敵更快 → 更多 killGold
利息（波末 currentGold × rate）
    ↓ 波次結束自動增加 currentGold
財富積累（下一波傷害更高）
```

→ 三者形成正向回饋閉環，但受以下自然限制：
- 升塔消耗 currentGold → 短暫斷開循環
- 防守壓力迫使玩家花錢 → 不可能純存錢不升
- 波次間距限制利息觸發頻率

## 四、LV4 → LV5 → LV6 梯度

| 等級 | 技能 | 設計意圖 |
|------|------|---------|
| LV4 | killGold（小）| 建立經濟習慣，讓玩家感受到「殺敵賺錢」|
| LV5 | killGold（中）+ 財富積累（低倍率）| 引入金幣↔傷害的動態關係 |
| LV6 | killGold（大）+ 財富積累（高倍率）+ 利息 | 完整經濟引擎解鎖，複利系統全開 |

---

## 五、純無塔最終完整定位

```
元素身份：「經濟引擎」

攻擊：中等基礎傷害，無弱點，動態加成（財富積累）
LV5：傷害開始跟金幣掛鉤，有錢 = 有力
LV6：+ 每波利息 + 更強的金幣↔傷害比例

選擇純無的理由：
✅ 不怕元素相剋，任何波次都穩定出力
✅ 每次擊殺自動賺錢，複利滾動
✅ LV6 後存錢越多傷害越高，利息持續增加金幣
✅ 與 LTW 系統深度協同：送兵賺 income + 無塔賺 killGold + 利息

不選純無的理由：
❌ 基礎 DPS 低於任何對應元素塔
❌ 需要前期投入（killGold + 存錢）才能發揮
❌ 被迫在「花錢升塔」和「存錢高傷害」之間取捨
```

## 六、待確認

2. **利息上限**：每波最多領幾金幣？（建議 30~50g，視整體 income 數值決定）
3. **利息歸屬**：純無塔的利息是玩家自己的 income bonus，還是額外 gold（不進入 income 計算）？
   - 建議：直接加 gold（不改 income），更簡單且不影響 LTW 平衡
4. **財富積累的傷害加成是否顯示在 UI**：玩家需要能看到「因為有錢所以現在傷害是 XXX」
