# Step 3：Lv4 全面評分 + score_adj 表

## 公式

```
DPS_score      = dmg × spd × (range/3.0) × (1 + aoe×0.7) × 2.841
raw_total      = DPS_score + Effect_score
effective_score = raw_total × score_adj
Lv4 目標分數  = 410（36 座平均值）
acceptable     = ±15%（348~472）
```

---

## 調整優先順序

1. **調效果公式**（已完成：killGold 降至 ×4，hpPct 加 Boss 加權）
2. **若仍偏高 → 調 DPS 參數**（改 damage/atkSpd）
3. **最終殘差 → 設定 score_adj**

## hpPct 機制注意事項

**建議在 towers.js/INFUSIONS 加入 `hpPctCap` 參數：**
```js
makeSkill('hpPct', { pct: 0.03, every: 3, cd: 1, cap: 120 })
// cap = 每次 proc 的最大傷害值（防止 Boss 被秒殺）
```

**需同步在 game.js doDmg/hpPct 處理加上 cap 判斷：**
```js
const rawHpDmg = enemy.hp * sk.pct;
const hpDmg = sk.cap ? Math.min(rawHpDmg, sk.cap) : rawHpDmg;
```

這是平衡問題，需要在 EXECUTE 時加入。
