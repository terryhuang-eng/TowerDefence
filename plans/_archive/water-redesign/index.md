# water-redesign — 問題分析與解法腦力激盪

## 問題陳述

水系（水水深寒、冰河塔等）能緩到底，但自身 DPS 殺不死——
然而任何傷害塔配上水系緩速，等效 DPS 大幅提升，造成：

1. **水系無法自成流派**（獨立跑純水輸）
2. **水系成為混搭王道**（水 + 任意高DPS = 最強組合）
3. **元素設計目標破功**（流派差異化無法實現）

---

## 根本原因分析

水系緩速的數學本質：
**讓敵人在射程內停留更長時間** → 等比例放大所有塔的有效 DPS

由於緩速對所有傷害來源一視同仁，水系扮演的角色是「全元素乘數器」而非獨立輸出流派。

目前 `深寒`（水水 Lv4）還附帶 `shred（碎甲 10%）`——
這進一步強化了「水讓別人更強」的支援定位，讓純水路線更難有立足點。

---

## 解法方向腦力激盪

### 方向 A：「寒傷（Frostbite）」— 冰冷層數造成持續水系傷害 ⭐ 推薦

**核心邏輯**：冰冷層數達到閾值後，敵人開始受「凍傷」——每秒損失 X% 最大HP 作為水系傷害。

```
chillStacks ≥ 20 層 → 每秒損失 (超出層數 × 0.05)% 最大HP（水系傷害）
上限：每秒 3% maxHP
```

例：38 層滿冰（超出 18 層）→ 每秒損失 0.9% maxHP，20 秒殺死無護甲目標。

**優點**：
- 純水自成 loop：冰冷 → 凍傷 → 死亡，不依賴其他元素
- 配合 freeze（定身）能讓凍傷持續累積，威力倍增
- 百分比傷害對 Boss 有意義（Boss 大量 HP 讓凍傷很重要）
- 不增強其他元素（凍傷是水系傷害，被火克制 ×0.7）

**缺點**：
- 百分比傷害可能對早期雜兵過強（設計時需給閾值）
- 數值平衡需測試

**配套調整**：移除 `深寒` 的 `shred`（碎甲），改為更純粹的冰冷疊層功能

---

### 方向 B：「裂冰（Frostbreak）」— 凍結狀態下一擊必殺

**核心邏輯**：冰河塔 Lv6 觸發 freeze（定身）時，若敵人 HP < 35%，立即破碎死亡。

```
freeze 觸發 + HP < 35% → 敵人破碎死亡（lv6 冰河專屬）
```

**優點**：
- 強烈的視覺特效感（freeze → shatter）
- 提供純水一個明確的殺敵方式

**缺點**：
- 依賴「先把血量打到 35%」——還是需要其他輸出協助
- 治標不治本：殺敵機制仍然依賴前置削血
- 只在 Lv6 生效，低等級水系依然無殺傷力

---

### 方向 C：「水刃（Hydroblade）」— 自身緩速直接加成自身傷害

**核心邏輯**：水系塔攻擊已被冰冷的目標時，傷害依目前緩速比例增加。

```
對 chillSlow > 0 的敵人：water傷害 × (1 + chillSlow)
例：80% 緩速 → 水系傷害 ×1.8
```

**優點**：
- 自我強化循環：水慢 → 水打更痛 → 殺得死
- 邏輯直觀：冰凍的敵人更脆弱

**缺點**：
- 也提升了其他元素（其他塔讓目標減速後，水系傷害也增加）
- 若移除「其他塔觸發此加成」的可能，邏輯複雜度提高
- 需要嚴格限定「只計算 water dmgType 攻擊」

---

### 方向 D：「冰護（Glacial Ward）」— 降低水系緩速對非水系的增益

**核心邏輯**：水系緩速對敵人造成「冰甲」效果——非水系傷害減少 15%（代表敵人凍成冰塊，護甲增加）。

```
chillStacks ≥ 10 → 非水系傷害 ×0.85
```

**優點**：
- 直接打擊「水 + 任意 DPS」的 combo 強度
- 鼓勵純水或水為主的建造

**缺點**：
- 懲罰混搭，但未必讓純水更強
- 玩家體驗差（感覺被限制而非獲得增益）
- 實作上每次傷害計算都要查 chillStacks，複雜度高

---

### 方向 E：「水系元素三角重設計」

目前：水克土 ×1.3，被火克 ×0.7
建議：將水的特殊克制改為「克制快速移動怪物」（如 charge/enrage 被動的怪）

**優點**：讓水在特定波次（速度型怪波）有強勢地位
**缺點**：需要重新設計大量怪物被動，工程量大，且不解決核心問題

---

## 推薦方向：A（寒傷）+ 移除深寒碎甲

### 設計願景

```
純水流派：
  大量冰冷塔 → 敵人快速達到 20+ 層冰冷
  → 凍傷開始 → 高層數時配合 freeze 定身讓凍傷持續
  → 百分比持續傷害殺死敵人

  特別強：對 Boss（大HP，凍傷傷害高）
  特別弱：快速過場的小波怪（來不及堆冰冷層）

水混搭流派（仍然可行，但不再無腦強）：
  失去碎甲 → 不再放大其他元素傷害
  凍傷是水系傷害 → 被火克制，不像「真實傷害」那麼通吃
```
